# Example GenBank Files

Place example GenBank files (`.gb`, `.gbk`, `.gbff`) in this directory to make them
available as demo inputs in the application.

## Where to Find Example Files

- [NCBI GenBank](https://www.ncbi.nlm.nih.gov/genbank/) — search for a bacterial genome
- PhiSpy test data in the [PhiSpy repository](https://github.com/linsalrob/PhiSpy)

## Suggested Examples

- A complete bacterial chromosome (e.g., *E. coli* K-12)
- A phage-containing genome with known prophages

> **Note:** Large files (> 50 MB) may be slow to process in the browser.
> For large batch analyses, use command-line PhiSpy.
